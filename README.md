# ShoppingPage
 路Java Servlets, CSS3, HTML5.     <br>
 路Display a list of products (read from a XML file).  <br>
 路You can buy articles, see your shopping cart and display your ticket.     <br>
 路Stocks are updated after buying. <br>
 
 # TO RUN THIS APP IN YOUR PC: 
-->Install Tomcat
-->Copy this file to Tomcat webapps directory
-->Turn on Tomcat
-->Access to localhost:yourport/ShoppingPage
-->Enjoy the web app!
